# summarize_variable ------------------------------------
the R version and the Python version have the same arguments:
df: 
in R, it needs to be a data.frame, in Python, it needs to be a Pandas
DataFrame
vars:
in R, gives a vector, in Python, gives a list
strata_var: 
only allows a single value 
sort: 
sort variables by their counts, default is False
digits:
default is 1
